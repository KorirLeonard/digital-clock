# Digital Clock

A functional digital clock built with HTML, CSS, and JavaScript that displays real-time hours, minutes, and seconds.

## 🕐 Live Demo

[View the Digital Clock](https://korirleonard.github.io/digital-clock/)

## Features

- Real-time clock display
- Shows hours, minutes, and seconds
- Clean and modern UI
- Responsive design
- Automatically updates every second

## Technologies Used

- **HTML5** — Structure
- **CSS3** — Styling and design
- **JavaScript** — Real-time clock functionality

## Project Structure

```
digital-clock/
├── index.html

├── script.js

```

## Getting Started

### View the Project

1. Visit the [live demo](https://korirleonard.github.io/digital-clock/)
2. Or clone the repository:

```bash
git clone https://github.com/KorirLeonard/digital-clock.git
cd digital-clock
```

3. Open `index.html` in your browser

## How It Works

The clock uses JavaScript's `Date` object to:
- Get the current time
- Update the display every second
- Format the time as HH:MM:SS
